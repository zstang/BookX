1. 光盘介绍

    本光盘是《国际大学生程序设计竞赛例题解（三）―― 图论、动态规划算法、综合题专集》的配套光盘，内容包括书中对应章节问题的测试数据，并附有程序源码，以便读者更好地学习。

2. 光盘内容

    光盘结构：光盘结构按照书本的目录对应编排，分别存放各章、节题目的数据（一节就是一道题目）的内容。

    数据文件：数据文件是本光盘的主要文件。数据文件分为输入文件和标准输出文件两类。一般来说，输入文件后缀名为.in或.dat, 标准输出文件的后缀名为.std。这些文件都是文本文件，可以用任何文本编辑器打开和编辑，文件内容的存储格式在书中有严格规范，它位于题目输入格式和输出格式部分。对于部分题目输出不惟一的情况, 则不一定提供标准输出文件,一般会附有测评程序的源代码。
  
    程序代码：书中的代码一般都用PASCAL语言编写，少数使用C++，它们的后缀名分别是pas和cpp。
    部分较长的程序已附在光盘对应章节的目录下。所有PASCAL程序都可以在Free Pascal v1.0.10下编译，而所有的C++程序都可以在         Microsoft Visual C++ 6.0下编译。编译时使用系统默认设置，无需引用其他类库。
    代码的编写遵循广泛认可的语言标准，尽量做到没有二义性，修改量少，甚至无需改动就可以用于其他的编译器或平台。


3. 光盘的使用

    光盘包含的数据文件, 可以使用任何平台上的任何文本编辑软件打开和编辑。由于书中已有某些标准程序代码, 因此光盘内不再提供。
    光盘没有附带应用程序，要得到可运行的代码必须配备适当的编译程序，推荐使用以下的编译器:
    * Free Pascal v1.0.10：这是开放源代码的PASCAL语言编译器，IDE和调试工具包。
    * Microsoft Visual C++ 6.0：这是微软公司(Microsoft)的程序编辑软件。
